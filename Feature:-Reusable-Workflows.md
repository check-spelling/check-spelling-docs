# Reusable Workflows

GitHub added [Reusable Workflows](https://docs.github.com/en/actions/using-workflows/reusing-workflows) in 2022.

## Thoughts

I looked into this in the summer of 2022 and found it didn't really reduce the size / effort for maintaining an action or a workflow, it seemed to just add to the complexity for releasing as it was one more moving part that had to be updated in a somewhat lock-step. That said. It _sounds_ appealing and there's definitely interest.
